# OrangeHRM Test Automation with Cypress

This project contains automated test scripts for OrangeHRM using Cypress, a modern end-to-end testing framework.

## 🚀 Features

- End-to-end testing of OrangeHRM functionalities
- Page Object Model (POM) implementation
- Custom commands for common actions
- Test data management
- HTML test reports generation

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm (Node Package Manager)
- OrangeHRM test environment
